# DecodeLabs Artificial Intelligence Tasks

This repository contains the projects and tasks I completed during my **Virtual Internship at DecodeLabs**. The projects focus on foundational Artificial Intelligence concepts, including rule-based systems, supervised learning, and recommendation systems. Each project helped strengthen my programming, machine learning, and problem-solving skills.

## Projects

### Project 1: Rule-Based Chatbot
Implemented a chatbot using if-else logic, dictionaries, and fallback responses.

### Project 2: Data Classification
Built a supervised machine learning model (KNN) to classify the Iris dataset using data preprocessing, train-test split, feature scaling, and evaluation metrics.

### Project 3: Recommendation System
Developed a content-based recommendation system using TF-IDF vectorization and cosine similarity to recommend items based on user preferences.
