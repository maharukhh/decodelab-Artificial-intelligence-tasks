Build a supervised machine learning model to classify Iris flower species using the K-Nearest Neighbors (KNN) algorithm. The project demonstrates the complete machine learning workflow, including data preprocessing, model training, evaluation, and visualization.

## 🧠 Project Overview

This project uses the well-known Iris dataset to classify flowers into three species based on their features.

The workflow includes:

- Loading the Iris dataset
- Data preprocessing
- Feature scaling
- Train-test split
- Training a KNN classifier
- Model evaluation
- Confusion matrix visualization
- K-value tuning to find the optimal number of neighbors
